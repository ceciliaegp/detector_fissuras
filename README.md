Enunciado desafio: 
Desenvolver, a partir das imagens e rótulos fornecidos pelo cliente, um sistema para detectar e identificar trincas e fissuras em paredes; implementar a solução em Python; preparar um relatório técnico que apresente o problema, a metodologia e os modelos utilizados para o desenvolvimento da solução e os resultados obtidos;
Extra: Implementar a solução na forma de uma aplicação (por exemplo, dentro de uma aplicação web); Permitir a execução do sistema com pouco recurso computacional em um dispositivo com baixa capacidade de processamento.
